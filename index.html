Overview

StackIt was designed with the goal to optimize front-end performances reducing the number of databases inserts.

The original idea was than instead of perform an insert from a front-end to store some trivial data, we can push them in a Redis stack then later, pull all the stacked data and insert them with a single query.

In this first version, StackIt is built as a Php library. But it's more in a prototyping way, this may evolve as I'm not sure than Php is the better language for this kind of library.

Installation

As for now it's still a Php library, the installation is really easy, you need only to

  • Download (or Fork/Clone) the library
  • Put it somewhere in your include path
